========================================================================================
                          RESOLV PIPELINE - SYSTEM CONTROL FLOW TOPOLOGY
========================================================================================

  The entire pipeline runs INSIDE one disposable per-issue Docker container. The host
  only launches it:

      docker run --rm --cap-add=SYS_ADMIN \
        -e RESOLV_GITHUB_TOKEN -e RESOLV_ANTHROPIC_API_KEY \
        resolv-sandbox:latest run --repo owner/name --issue 123

  CLI users run that directly; the webhook server runs it per queued issue.
  CodeRabbit is NOT a pipeline node — it reviews the pushed PR in the cloud afterwards.
